# Get bucket info

Get meta information of a bucket. The information includes bucket type and column definitions.

**URL** : `/api/v1/{username}/{bucketname}?delete`

**Method** : `DELETE`

**Auth required** : YES

**Permissions required** : Owner

**Data constraints**

N/A

**Data examples**

```
GET /api/v1/admin/testtimeseriesbucket?meta
```
